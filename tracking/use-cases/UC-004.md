# UC-004: Composable Re-Ranking Pipeline

## Status: SPEC'D

## Objective Alignment
Links to: OBJ-001 ([Build a Production-Quality Re-Ranking Library](../objectives/OBJ-001.md))

## Description
Provide a builder-pattern pipeline that wires together a tokenizer, one or more scorers, and an aggregator to produce ranked results. This is the user-facing API — the primary way library consumers interact with the re-ranking engine.

## Beneficiaries
- Library users: intuitive API for building custom ranking pipelines
- Application developers: composable scoring without manual orchestration
- Agent developers: clear API surface to implement against
