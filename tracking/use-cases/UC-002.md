# UC-002: Text Tokenization

## Status: IMPLEMENTED

## Objective Alignment
Links to: OBJ-001 ([Build a Production-Quality Re-Ranking Library](../objectives/OBJ-001.md))

## Description
Provide a standard tokenizer that splits document text into lowercase tokens, removes punctuation and stopwords, and produces positioned Token objects. This is the first stage of the re-ranking pipeline — all scorers operate on tokens, not raw text.

## Beneficiaries
- Scorer implementations: receive clean, normalized tokens
- Library users: sensible default tokenization without configuration
- Pipeline builders: pluggable tokenizer interface for custom strategies
