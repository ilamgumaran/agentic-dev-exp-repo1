# UC-003: BM25 Scoring

## Status: SPEC'D

## Objective Alignment
Links to: OBJ-001 ([Build a Production-Quality Re-Ranking Library](../objectives/OBJ-001.md))

## Description
Implement the BM25 scoring algorithm as the primary token-matching scorer. BM25 is the industry-standard baseline for text retrieval, improving on TF-IDF with document length normalization and term frequency saturation. Must support multi-field documents with configurable field weights.

## Beneficiaries
- Library users: production-grade baseline scorer out of the box
- E-commerce search: product catalog ranking by keyword relevance
- Financial search: record matching with precise term weighting
- Pipeline builders: composable with semantic scorers via aggregators
