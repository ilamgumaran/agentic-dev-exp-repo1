# UC-007: Character N-Gram Cosine Similarity Scorer

## Status: IMPLEMENTED

## Objective Alignment
Links to: OBJ-001 ([Build a Production-Quality Re-Ranking Library](../objectives/OBJ-001.md))

## Description
Provide a lightweight, zero-dependency semantic-ish scorer using cosine similarity over character n-gram frequency vectors. Produces bounded [0,1] scores — deliberately a different scale from BM25's unbounded scores — making it the second signal needed to demonstrate and test hybrid rank fusion. Catches morphological matches ("running" ~ "runner") that token-exact BM25 misses, without requiring embeddings.

## Beneficiaries
- **Hybrid search**: the second scorer that makes fusion (UC-005) meaningful
- **DIY & e-commerce shoppers**: fuzzy/morphological matching for approximate queries
- **Library users**: a semantic-ish signal with zero dependencies and no model hosting
